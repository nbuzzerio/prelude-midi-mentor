# Prelude: MIDI Mentor

An open-source MIDI-powered piano sight-reading trainer built with Next.js and TypeScript.

## Vision

Prelude helps musicians develop fluent music reading using a real MIDI keyboard.

The long-term goal is to provide an enjoyable, focused alternative to traditional flashcards by combining:

- 🎹 MIDI keyboard input
- 🎼 Interactive staff notation
- 📖 Sight-reading drills
- 🎵 Music theory exercises
- 📈 Progress tracking

## MVP Goals

- Bass clef note recognition
- Treble clef note recognition
- Grand staff mode
- MIDI keyboard input
- Instant feedback
- Statistics

## Planned Features

- Chords
- Scales
- Intervals
- Rhythm
- Ear training
- Composer exercises
- AI practice coach

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Web MIDI API
- VexFlow
